# Excel to PDF 変換ツール

Excelファイルを読み込み、氏名欄に名前と電子印を追加してPDFに変換するStreamlitアプリケーションです。

## 機能

- Excelファイルのアップロード
- 氏名を追加する列の選択
- 追加する名前の入力
- 電子印の色のカスタマイズ
- 電子印のプレビュー表示
- PDFへの変換とダウンロード

## 実行方法

### パッケージ版（推奨）

1. `streamlit_excel_to_pdf_package.zip`をダウンロードして解凍します
2. 解凍したフォルダに移動します
3. 実行権限を付与します: `chmod +x run_app.sh`
4. スクリプトを実行します: `./run_app.sh`

初回実行時は必要なパッケージがインストールされ、その後Streamlitアプリケーションが起動します。

### 直接実行

必要なパッケージがインストールされている場合は、以下のコマンドで直接実行できます：

```bash
streamlit run app.py
```

## 必要なパッケージ

- streamlit
- pandas
- openpyxl
- reportlab
- pillow

## サンプルデータ

`sample_data.xlsx`にサンプルデータが含まれています。このファイルをアプリケーションにアップロードしてテストできます。
